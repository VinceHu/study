### 🎯 口语化回答的重要性

**为什么需要口语化回答？**

1. 面试是口头交流，不是背书
2. 帮助候选人组织语言，避免紧张时不知道怎么说
3. 展示自然的表达方式，更有说服力

**口语化回答的特点：**

- 使用"首先"、"简单来说"、"具体实现上"等过渡词
- 语气自然，像在和面试官聊天
- 有逻辑层次，从概念到原理到实践
- 适当使用"我理解的是"、"在实际项目中"等个人化表达

#### 步骤3：更新 VitePress 配置

编辑 `.vitepress/config.mts`，在对应的侧边栏添加：

```typescript
{
  text: '分类名',
  items: [
    // ... 其他题目
    { text: '你的题目', link: '/questions/分类/你的题目' }
  ]
}
```

#### 步骤4：更新知识点索引

编辑 `knowledge-map/index.md`，添加题目链接：

```markdown

```

#### 步骤5：更新首页统计

编辑 `index.md`，更新题目数量。

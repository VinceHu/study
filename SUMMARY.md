## 🚀 如何使用

### 本地开发

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run docs:dev

# 3. 访问 http://localhost:5173
```

### 构建生产版本

```bash
# 构建
npm run docs:build

# 预览
npm run docs:preview
```

### 部署到线上

#### 方式一：GitHub Pages（推荐）
1. 推送代码到 GitHub
2. 启用 GitHub Pages（选择 GitHub Actions）
3. 自动部署

#### 方式二：Vercel
1. 导入 GitHub 仓库
2. 自动识别并部署

#### 方式三：Netlify
1. 连接 GitHub 仓库
2. 配置构建命令和输出目录
3. 部署

详见 [DEPLOYMENT.md](DEPLOYMENT.md)
